# IMDb Sentiment Classification using Deep Learning 
• Built LSTM and BiLSTM models in PyTorch for IMDb sentiment classification (25K train / 25K
test), achieving 80.93% and 83.91% test accuracy via Optuna-based hyperparameter tuning

• Fine-tuned BERT and RoBERTa Transformers, achieving 90.03% and 93.40% accuracy respectively

• Achieved 12.5% accuracy improvement transitioning from recurrent to transformer architectures
